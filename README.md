# Game Project

这是一个使用 HTML、CSS 和 JavaScript 构建的简单游戏项目。玩家可以在游戏中探索、挖掘资源、与怪物战斗，并尝试生存下去。

## 项目特点

- **游戏机制**：包括玩家移动、挖掘方块、怪物生成与战斗、计时器、生命值管理等。
- **资源与道具**：支持多种方块类型（如钻石矿、TNT、水等）和怪物类型（如僵尸、苦力怕）。
- **音效与动画**：为各种操作（如爆炸、拾取物品、受伤）添加了音效支持。
- **响应式界面**：通过 CSS 实现美观的界面和交互体验。

## 文件结构

- `index.html`：游戏的主界面结构。
- `style.css`：负责游戏界面的样式和视觉效果。
- `script.js`：实现游戏逻辑和交互功能。

## 如何运行

1. 将项目文件下载到本地。
2. 使用浏览器打开 `index.html` 文件即可开始游戏。

## 控制方式

- 使用方向键或 WASD 键控制玩家移动。
- 靠近方块后按空格键进行挖掘。
- 避免怪物攻击，收集资源以提升生存能力。
