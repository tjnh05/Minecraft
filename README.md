# Minecraft矿洞探险游戏

这是一个基于HTML、CSS和JavaScript开发的Minecraft风格矿洞探险游戏。玩家可以在15x15的游戏板中探索、挖掘资源、与怪物战斗，并在限定时间内尽可能多地收集钻石资源。

## 游戏特色

- **多种方块类型**：包括钻石矿、TNT、岩浆、水、玻璃等12种不同方块，每种都有独特功能
- **两种怪物类型**：普通怪物、僵尸，每种都有独特的行为模式
- **完整方块图鉴**：内置帮助系统，详细展示所有方块和怪物的外观与功能
- **难度系统**：提供简单、中等、困难三种难度，影响时间限制和游戏机制
- **道具系统**：钻石剑、金苹果、TNT等道具增强游戏体验
- **音效支持**：为各种游戏操作添加音效反馈
- **实时战斗系统**：玩家与怪物的动态战斗机制
- **响应式设计**：自适应不同屏幕尺寸，完美支持移动设备
- **帮助弹窗系统**：简洁界面的帮助按钮，提供完整的游戏说明和操作指南
- **自定义弹窗**：美观的游戏内提示和确认对话框
- **动态UI反馈**：生命值和剩余时间根据状态变化颜色，提供视觉警告
- **优化布局**：界面元素与游戏板宽度自适应，确保视觉一致性
- **纪录系统**：自动保存和显示最快胜利时间，激励玩家挑战自我
- **玩家标识**：醒目的绿色脉动动画和表情符号，轻松定位玩家位置

## 文件结构

- `index.html`：游戏的主界面结构和UI布局
- `style.css`：游戏界面样式、动画效果和响应式设计
- `script.js`：实现游戏核心逻辑、交互功能和音效系统
- `favicon.ico`：游戏图标

## 游戏玩法

### 基本操作
- **键盘控制**（PC/笔记本）：方向键或WASD键移动到相邻方块，空格键挖掘当前方块
- **触屏控制**（手机/平板）：📱点击相邻方块移动，长按玩家位置挖掘
- **道具使用**：数字键1-5使用对应道具
- **玩家识别**：😊笑脸带有绿色脉动光晕，轻松定位玩家位置
- **游戏帮助**：点击"❓ 游戏帮助"按钮查看详细说明和方块图鉴

### 游戏目标
- 在限定时间内消灭所有怪物获得胜利
- 收集钻石资源提升分数
- 避免被怪物攻击或挖掘岩浆（立即死亡）
- 合理使用道具增强生存能力
- 挑战最快胜利纪录，不断超越自我

### 方块类型说明
- **岩浆**：橙红色方块，挖掘立即导致游戏结束
- **钻石矿**：需要挖掘两次才能获得钻石
- **金苹果**：恢复5点生命值
- **TNT/反物质TNT**：爆炸物，挖掘会减少生命值
- **水**：蓝色方块，无法移动进入
- **其他方块**：石头、木头、圆石、玻璃、泥土等基础方块

### 难度设置
- **简单模式**：3分钟时间限制，金苹果恢复效果100%
- **中等模式**：2分钟时间限制，金苹果恢复效果100%
- **困难模式**：1分钟时间限制，金苹果恢复效果67%

## 技术实现

- **前端框架**：纯HTML5、CSS3、JavaScript (ES6+)
- **游戏引擎**：自研轻量级游戏引擎
- **响应式布局**：基于Flexbox的CSS设计
- **音效系统**：Web Audio API生成的动态音效
- **状态管理**：集中式游戏状态管理
